Gesture - Hand Recognition Application

Gesture is a desktop application that allows the user the use of hand gestures to control user-specified applications and certain operating system features. This project was managed by the Aggie Coding Club and was progressed on in Spring 2021 leading into Summer 2021. This was a large introduction to modern technologies for most members of the team, including myself, but provided a great learning experience for everyone. I worked on both the frontend and backend of the application. Some examples that I completed were, setting up the backend web server using Flask as our python framework, multiple UI features & styling, setting up navigation within the UI, and touching up the OpenCV logic to assure the most efficiency when tracking was live as the consistent detection had the potential for high CPU usage.

Frontend: React, Electron
Backend: Python, Flask, OpenCV, SQLite

Dynata UFS Finance Reporting App

This project was the main project I worked on during my Summer 2021 internship at Dynata. This project consisted mainly of a user interface that would allow the Dynata Finance team the ability to generate fulfillment reports along with fulfillment audits. Previously the director of software engineering would have to manually generate a report from the UFS dataset, but this user interface would allow for the gap to be closed between the Finance team and the UFS data. The project was a massive learning experience for me. It was my first step into a large corporation's development cycle which led to me experiencing different methodologies and practices along the way. The project was overseen by a Senior Software Engineer who assisted in solving complications I had and provided me code review when needed. The project means a lot to me as it will be continuously used over Dynata's lifetime and offered me an invaluable learning experience.

Frontend: React, Material UI, React Testing Library, Jest
Backend: Java, Groovy, Spring Boot, Spock, Amazon Aurora (MySQL)

Orion - Virus Protection Software

Orion is a desktop virus protection application. It provides users with the ability to run specified or full system virus scans, various backup options, and a fully integrated password manager. This project was my first deep dive into a larger personal project. I had always been fascinated by desktop user interfaces and wanted to take a stab at creating something different. I designed all UX for the interface and performed all development for both front and backend. I was immensely pleased with how everything came out, and even still use it to this day to run quick scans. As far as technology goes, the interface was created with a technology called Qt Creator, a popular desktop application framework for Python and C++. The actual virus detection is done with a library for Python, Yara. The application did contain a smaller backend web server for storing and encrypting password information from the frontend application to provide an extra level of protection to your information. Overall, this was a great experience and provided an awesome turnout.

Frontend: Python, PyQt5
Backend: Python, Flask, Yara, SQLite
